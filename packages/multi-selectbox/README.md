# react-native-multi-selectbox

Publishable library workspace (**2.0.0-alpha** during migration).

SelectBox JS was copied from legacy `/lib` in **Phase 2** so the Expo example can run against the workspace package. **Phase 3** formalizes the move (remove `/lib` dual source); **Phase 4** adds full TypeScript types.

## Peers

- `react`, `react-native`
- `react-native-svg` (use `npx expo install react-native-svg` in Expo apps)
- `lodash` (used by SelectBox filters / helpers)
